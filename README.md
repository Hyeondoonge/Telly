# Telly

## 소개
spa기반의 TodoList 애플리케이션 Telly.

***
## 설치

Server(npm version(7.10.0))
```
npm install
```

Client
```
npm install
```

***
## 시작

Server
```
node app
```
***
## 기술 스택
### 언어
- javascript(ES6)

### 프레임 워크
- koa

### db
- mongoDB

### 그 외
- RESTfulAPI
- Hash Routing

***
## 주요 기능
1. 프로젝트 생성
2. D & D를 이용한 작업 상태 변경
3. 중요한 작업 마킹
