# Telly

spa기반의 TodoList 애플리케이션

## Setup

Server(npm version(7.10.0))
```
npm install
node app
```

Client
```
npm install
```

## Technologies
### language
- javascript(ES6)

### framework
- koa

### db
- mongoDB

### others
- RESTfulAPI
- Hash Routing


## Features
1. 프로젝트 생성
2. D & D를 이용한 작업 상태 변경
3. 중요한 작업 마킹
